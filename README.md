# Chat Application 
Chat Application in Angular 8, Asp.net Core 2.2.0, Signal R 1.1.0

### Prerequisite 
.Net Core 2.2.0<br />
Nodejs 10.15.3<br />
Angular CLI 8.0 <br />

### Setup Project
git clone https://github.com/AngularExamplesHub/angular-chat-app.git

### install npm packages and restore dotnet nuget pakages in command promt:
npm install <br />
dotnet restore  

### run project
dotnet run
